Abrir el programa:
De preferencia el programa debe ser ejecutado desde consola de Windows o Linux "python ubicación/limpiador.py" sin comillas.
esto debido a que ejecutarla desde la terminal de Python no muestra colores en el texto
Ejecución del programa:
Una vez en ejecución automáticamente busca los documentos en la carpeta ‘clientes’ que se encuentre en la misma ubicación que el programa. 
Luego de realizado esto muestra una lista con los documentos escritos "correctamente" y los que presentan algún "error". 
Finalmente consulta si desea reparar los documentos con errores aceptando como respuesta a la pregunta: 's', 'S' que significan si, 
reparando los documentos con errores y mostrando una lista con los documentos que se modificaron  y 'n', 'N' que significan no y muestra un
 mensaje indicando que no se modificó ningún documento.
